# Admin Specification: AI Audit Logs (Deep Analysis & Compliance)

## ১. ওভারভিউ (Overview)
অ্যাডমিন প্যানেলের **"AI অডিট লগস"** পেজটি শুধুমাত্র রিয়েল-টাইম প্রম্পট দেখার জন্য নয়, বরং এটি প্ল্যাটফর্মের এআই ব্যবহারের **ডিপ অ্যানালাইসিস, কমপ্লায়েন্স ট্র্যাকিং এবং কস্ট অপটিমাইজেশন (Cost Optimization)**-এর মূল কেন্দ্র। এন্টারপ্রাইজ বা ইন্ডাস্ট্রি স্ট্যান্ডার্ড অনুযায়ী, যেকোনো এআই ইন্টিগ্রেশনে সিকিউরিটি, ডেটা প্রাইভেসি এবং বিলিং অডিট থাকা বাধ্যতামূলক। এই পেজটি সেই গ্যাপ পূরণ করবে।

---

## ২. পেজের মূল সেকশনসমূহ (Key Sections)

এই পেজে প্রধানত ৪টি ট্যাব বা সেকশন থাকবে:

### ২.১. ডিপ অ্যানালিটিক্স ও কস্ট ড্যাশবোর্ড (Analytics & Cost Dashboard)
এটি একটি অ্যাডভান্সড ড্যাশবোর্ড, যা মূলত বিলিং এবং ইউজ প্যাটার্ন নিয়ে কাজ করবে।
- **টোকেন ইউসেজ ট্রেন্ড (Token Usage Trend):** গত এক সপ্তাহ, মাস বা নির্দিষ্ট ডেট-রেঞ্জে প্রতিদিন বা প্রতি ঘণ্টায় কত টোকেন খরচ হয়েছে তার লাইন চার্ট (Line Chart)। 
- **কস্ট এস্টিমেশন (Cost Estimation):** বর্তমান API রেট অনুযায়ী এই মাসে আনুমানিক কত ডলার বিল আসতে পারে (Estimate vs Budget) তার হিসাব।
- **মডেল পারফরম্যান্স (Model Performance):** এআই-এর Response Time (গড়ে কত সেকেন্ড লাগছে) এবং Error Rate (e.g., 500 Internal Server Error, API Quota Exceeded) এর রিয়েল-টাইম গ্রাফ।
- **ভৌগোলিক ও ডেমোগ্রাফিক হিটম্যাপ (Geo-Demographic Heatmap):** বাংলাদেশের কোন জেলা থেকে কোন রোগের জন্য সবচেয়ে বেশি প্রম্পট বা স্ক্যান আসছে, তার ম্যাপ ভিউ বা বার চার্ট।

### ২.২. ইনভেস্টিগেশন ও ফিল্টারিং ইঞ্জিন (Investigation & Filtering Engine)
সাধারণ সার্চের বাইরে এটি একটি শক্তিশালী ফিল্টার বা Query Builder হবে, যা দিয়ে হিস্টোরিক্যাল ডেটা খনন করা যাবে।
- **অ্যাডভান্সড ফিল্টারস (Advanced Filters):**
  - **Date Range:** কাস্টম তারিখ অনুযায়ী।
  - **User/Farmer ID:** নির্দিষ্ট কৃষকের করা সব হিস্টোরি।
  - **Type:** Text, Voice, Image Scan।
  - **Entity/Topic:** CropName (যেমন: ধান, আলু), Disease Name (যেমন: লেট ব্লাইট, ব্লাস্ট), Action (যেমন: সার, সেচ)।
  - **Status:** Success, Low Confidence, Model Error, Token Limit Exceeded।
- **অ্যানোমালি ডিটেকশন (Anomaly Detection):** যদি একই ইউজার অল্প সময়ে অনেক বেশি রিকোয়েস্ট (Rate Limiting/Abuse) পাঠায়, তবে সিস্টেম তাকে ফ্ল্যাগ (Flag) করবে এবং অ্যাডমিন এখানে সেই ইউজারের লিস্ট দেখতে পাবেন।

### ২.৩. সিস্টেম ও সিকিউরিটি অডিট (System & Security Audit)
এন্টারপ্রাইজ স্ট্যান্ডার্ডে "Who did what and when?" অত্যন্ত জরুরি। সুপার-অ্যাডমিনদের কাজের ট্র্যাক রাখা হবে এখানে।
- **অ্যাডমিন অ্যাক্টিভিটি লগ (Admin Action Logs):**
  - **API Key Change:** কবে, কখন, কোন অ্যাডমিন Gemini বা Vectorize-এর API Key পরিবর্তন করেছে।
  - **System Prompt Update:** আগের সিস্টেম প্রম্পট কী ছিল এবং কে তা এডিট করে নতুন কী সেভ করেছে (Diff viewer)।
  - **Knowledge Base Update:** RAG ডাটাবেসে নতুন কোন PDF আপলোড বা ডিলিট করা হয়েছে তার রেকর্ড।
- **টাইমস্ট্যাম্প ও আইপি ট্র্যাকিং (Timestamp & IP Logs):** প্রতিটি সিকিউরিটি পরিবর্তনের সোর্স IP এবং ইউজার এজেন্ট (ব্রাউজার/ডিভাইস)।

### ২.৪. কমপ্লায়েন্স ও রিপোর্ট জেনারেশন (Compliance & Export)
স্টেকহোল্ডার (বোর্ড অফ ডিরেক্টরস বা ইনভেস্টর) অথবা সরকারি ডেটা অডিটের জন্য রিপোর্ট তৈরির অপশন।
- **পিডিএফ/ম্যানেজমেন্ট রিপোর্ট (Export to PDF/Excel):** এক ক্লিকে গত মাসের "AI Performance & Cost Report" জেনারেট করা।
- **ডেটা প্রাইভেসি মাস্কিং:** এক্সপোর্ট করা রিপোর্টে কৃষকের ব্যক্তিগত মোবাইল নম্বর বা পরিচয় গোপন (e.g., 01712***890) রাখার টগল অপশন (Data Anonymization)।

## ৩. ইউআই/ইউএক্স (UI/UX) আউটলাইন
- **লেআউট:** ফুল-উইডথ ড্যাশবোর্ড, উপরে ফিল্টার প্যানেল এবং নিচে হাই-ডেনসিটি (High-density) ডেটা টেবিল।
- **কালার কোডিং:** Error বা Abuse বোঝাতে Red, Warning (High cost/Low confidence) বোঝাতে Orange, এবং স্বাভাবিক ফ্লো-র জন্য Green বা Neutral কালার।
- **মডাল/স্লাইড-ওভার:** যেকোনো নির্দিষ্ট লগের ওপর ক্লিক করলে একটি স্লাইড-ওভার প্যানেলে ওই রিকোয়েস্টের পুরো JSON বডি (Raw Input & Output JSON) দেখা যাবে ডেভেলপারদের ডিবাগিং সুবিধার জন্য।

---

## ৪. ব্যাকএন্ড API এর রূপরেখা (Backend APIs Needed)

এই পেজটির জন্য নিচের API এন্ডপয়েন্টগুলো লাগবে:
- `GET /api/admin/audit/costs` - কস্ট এবং টোকেনের ড্যাশবোর্ড ডেটা (Aggregate and Group by Date/Region)।
- `GET /api/admin/audit/logs` - অ্যাডভান্সড ফিল্টারিং প্যারামিটার সহ হিস্টোরিক্যাল ডেটা ফেচ (Pagination & Filtering)।
- `GET /api/admin/audit/security` - অ্যাডমিনদের পলিসি ও সিস্টেম চেঞ্জ লগস।
- `POST /api/admin/audit/export` - ব্যাকগ্রাউন্ড জব হিসেবে বড় ডেটাসেট CSV/PDF এ এক্সপোর্ট করার রিকোয়েস্ট।

## ৫. এআই ইঞ্জিন বনাম এআই অডিট লগস (পার্থক্য)
- **এআই প্রম্পট ইঞ্জিন:** এটি হলো **অপারেশনাল কন্ট্রোল রুম**। এখান থেকে রুলস সেট করা হয়, API Key বসানো হয় এবং লাইভ ট্রাফিক দেখা হয় যে সিস্টেম কাজ করছে কিনা।
- **এআই অডিট লগস:** এটি হলো **ডেটা অ্যানালাইসিস এবং সিকিউরিটি সেন্টার**। গত মাসের খরচ কত হলো, কোন এলাকার মানুষ কী নিয়ে ভুগছে, সিস্টেমে কেউ অ্যাবিউজ করছে কিনা, তার গভীরে গিয়ে অ্যানালাইসিস করার জায়গা।
